# Research Paper Classification and Recommendation System

## Overview

A web-based system that classifies research topics using TF-IDF and Decision Tree Machine Learning and recommends relevant research papers.

## Technologies Used

- React.js
- Node.js
- Express.js
- Python
- Scikit-learn
- TF-IDF
- Decision Tree

## Features

- Research topic classification
- Research paper recommendation
- NLP-based processing
- Full-stack web application
